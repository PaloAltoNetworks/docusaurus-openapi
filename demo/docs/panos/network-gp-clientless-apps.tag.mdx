Manages a list applications that are accessible through the GlobalProtect Clientless VPN. GlobalProtect
Clientless VPN provides secure remote access to common enterprise web applications that use HTML, HTML5,
and JavaScript technologies. Users have the advantage of secure access from SSL-enabled web browsers
without installing GlobalProtect software.
