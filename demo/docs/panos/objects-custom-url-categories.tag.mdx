Manages custom URL categories. Use custom URL categories to create a custom list of URLs and use it in a URL filtering profile or as
match criteria in policy rules. In a custom URL category, you can add URL entries individually, or import a text file that contains a list of URLs.




```mdx-code-block
import DocCardList from '@theme/DocCardList';
import {useCurrentSidebarCategory} from '@docusaurus/theme-common';

<DocCardList items={useCurrentSidebarCategory().items}/>
```
