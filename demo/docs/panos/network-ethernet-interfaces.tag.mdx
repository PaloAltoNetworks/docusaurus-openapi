Manages Ethernet interfaces. You can configure Ethernet interfaces as tap, high availability (HA), log card
(interface and subinterface), decrypt mirror, virtual wire (interface and subinterface), Layer 2
(interface and subinterface), Layer 3 (interface and subinterface), or aggregate Ethernet. The available
interface types and transmission speeds vary by hardware model.
