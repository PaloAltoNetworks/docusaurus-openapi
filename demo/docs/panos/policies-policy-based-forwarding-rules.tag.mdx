Manages Policy-Based Forwarding (PBF) rules. A PBF rule allows you to direct traffic to a specific egress interface
on the firewall and override the default path for the traffic.

When creating a PBF rule, you must specify a name for the rule, a source zone or interface, and an egress interface.
