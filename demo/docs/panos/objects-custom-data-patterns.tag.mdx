Manages custom data patterns that enable you to filter for sensitive information. Custom data patterns enable you to match on senstive contents
or specific metadata in a file. You can select from a predefined list of patterns, specify regular expressions, or select the file properties
that you want to match on and enforce Data Filtering policies.
