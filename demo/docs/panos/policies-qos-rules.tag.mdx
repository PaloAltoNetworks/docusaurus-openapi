Manages Quality of Service (QoS) rules. These rules define the traffic that receives QoS treatment. Each
rule also identifies a Quality of Service class (numbered 1 - 8) using the `action` field in the
rule payload. QoS classes are defined on the firewall using QoS Policy rules.
