Manages VLANs. The firewall supports VLANs that conform to the IEEE 802.1Q standard. Each
Layer 2 interface defined on the firewall can be associated with a VLAN. The same VLAN can be
assigned to multiple Layer 2 interfaces but each interface can belong to only one VLAN.
