Manages SD-WAN Traffic Distribution profiles, which specify how the firewall determines a new best path if the current preferred path fails a
path quality requirement. An SD-WAN Traffic Distribution profile includes Link Tags, which the distribution method uses to narrow its selection
of a new path.
