Manages DoS Protection security profiles used to protect against new session IP flood attacts and provide resource protection (maximum concurrent
session limits for specified enpoints and resources). DoS Protection profiles protect specific devices (classified profiles) and groups
 of devices (aggregate profiles) against SYN, UDP, ICMP, ICMPv6, and other IP flood attacks.
