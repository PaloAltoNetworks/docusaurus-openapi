Manages virtual SD-WAN interfaces. A virtual SD-WAN interface is a VPN or DIA group of one or more
interfaces that constitute a numbered, virtual SD-WAN Interface to which you can route traffic. The
paths belonging to the SD-WAN Interface all go to the same destination WAN.
