## Set allowed actions for an enterprise



Sets the actions that are allowed in an enterprise. To use this endpoint, the enterprise permission policy for `allowed_actions` must be configured to `selected`. For more information, see &quot;[Set GitHub Actions permissions for an enterprise](#set-github-actions-permissions-for-an-enterprise).&quot;

You must authenticate using an access token with the `admin:enterprise` scope to use this endpoint.
