## Get the audit log for an organization



**Note:** The audit log REST API is currently in beta and is subject to change.

Gets the audit log for an organization. For more information, see &quot;[Reviewing the audit log for your organization](https://docs.github.com/github/setting-up-and-managing-organizations-and-teams/reviewing-the-audit-log-for-your-organization).&quot;

To use this endpoint, you must be an organization owner, and you must use an access token with the `admin:org` scope. GitHub Apps must have the `organization_administration` read permission to use this endpoint.
