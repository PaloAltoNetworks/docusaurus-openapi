## Add organization access to a self-hosted runner group in an enterprise



Adds an organization to the list of selected organizations that can access a self-hosted runner group. The runner group must have `visibility` set to `selected`. For more information, see &quot;[Create a self-hosted runner group for an enterprise](#create-a-self-hosted-runner-group-for-an-enterprise).&quot;

You must authenticate using an access token with the `admin:enterprise` scope to use this endpoint.
